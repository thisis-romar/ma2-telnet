## Standard CD Command Forms
- `CD 5`  
  - Element-index: Go to element 5
- `CD "Sequence 1"`  
  - Element name: Go to Sequence named "Sequence 1"
- `CD Group 10`  
  - Object-type/Object-ID: Go to Group 10
- `CD ..`  
  - Parent directory: Go up one level
- `CD /`  
  - Root directory: Go to root
